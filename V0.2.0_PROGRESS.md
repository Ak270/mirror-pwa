## 💡 Implementation Notes

### Groq Rate Limiting
- Uses in-memory Map (resets on serverless function cold start)
- Limits: 50 checkins/day, 10 insights/day per user
- Gracefully falls back to static copy when rate limited

### iOS Notification Fix
- Detects iOS standalone mode vs Safari browser
- Shows install banner if not in standalone mode
- Tracks subscription source for debugging
- Verifies subscriptions every 7 days

### Slip Note UX
- Bottom sheet animation for mobile-first experience
- Optional field - users can skip
- Helps identify patterns in slip behavior
- Non-judgmental language throughout

### Dark Mode
- Loads theme before page render (no flash)
- Persists to localStorage
- CSS variables for easy theming
- Vault PIN screen excluded from dark mode (has own aesthetic)
