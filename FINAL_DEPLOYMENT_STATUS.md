## 📋 Post-Deployment Checklist

### 1. Test AI Notifications
```bash
# After deployment, test the endpoint
curl -X POST \
  -H "Authorization: Bearer YOUR_CRON_SECRET" \
  https://mirror-pwa.vercel.app/api/notifications/send
```

**Expected**: `{"sent": 0, "errors": []}` (if no reminders due now)

### 2. Set Up Habit Reminder
1. Go to Mirror app
2. Create/edit a habit
3. Set reminder for 5 minutes from now
4. Don't log it today
5. Wait for notification
6. Should receive AI-generated message

### 3. Update Widget
1. Download new `MirrorWidget.js` from `/public/MirrorWidget.js`
2. Copy entire script
3. Open Scriptable app
4. Replace old widget script
5. Get fresh API token from Profile page
6. Update `API_TOKEN` in script
7. Widget should show dark UI

### 4. Fix Token Expiry (Optional but Recommended)
1. Go to Supabase Dashboard
2. Authentication → Settings
3. JWT Expiry: Change from `3600` to `2592000` (30 days)
4. Save
5. Get fresh token from Mirror app
6. Update widget script
