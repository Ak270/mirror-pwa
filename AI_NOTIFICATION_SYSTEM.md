## How It Works

### Notification Timing Phases

**15 minutes before** (`early_reminder`):
- Gentle heads-up
- Preparation suggestions
- Example: "Hey buddy, get ready for the gym in 15!"

**10 minutes before** (`prep_reminder`):
- Get ready, gather items
- Example: "Time to grab your gym clothes 👟"

**5 minutes before** (`final_reminder`):
- Final push, almost time
- Example: "Almost time! Get pumped, we start in 5 🔥"

**0 minutes** (`time_now`):
- Action-oriented, let's go
- Example: "Let's go! Time to crush that workout 💪"

### AI Generation

Each notification is generated by Groq AI (Llama 3.3 70B) with:
- **Habit name**: "Exercise", "Read", "Meditate"
- **Current streak**: Mentions protecting streaks > 7 days
- **Timing phase**: Adjusts tone based on urgency
- **Temperature**: 0.8 for creative variety
- **Max tokens**: 50 (keeps messages short)

### Example Notifications

**Exercise habit at 10:50 AM**:
- **10:35** (15 min): "Hey buddy, get ready for the gym! 💪"
- **10:40** (10 min): "Time to grab your gym clothes and water bottle"
- **10:45** (5 min): "Almost time! Get pumped, we start in 5 🔥"
- **10:50** (0 min): "Let's go! Time to crush that workout 💪"

**Reading habit with 12-day streak**:
- **15 min**: "Your 12-day streak is waiting! Grab your book 📚"
- **10 min**: "Find a cozy spot, reading time soon"
- **5 min**: "5 minutes until you protect that 12-day streak!"
- **0 min**: "Time to read! Keep that streak alive 🔥"

**Meditation habit**:
- **15 min**: "Peace is 15 minutes away 🧘"
- **10 min**: "Find your meditation spot, almost time"
- **5 min**: "5 minutes to calm. Take a deep breath"
- **0 min**: "Time to meditate. Let's find your center 🕉️"

## Technical Implementation

### Notification Payload

```javascript
{
  title: "Exercise",  // Habit name (not "Mirror")
  body: "Hey buddy, get ready for the gym! 💪",  // AI-generated
  icon: '/icons/icon-192.svg',
  badge: '/icons/badge-72.svg',
  url: `/log?habit_id=${habit.id}`,
  tag: `habit-reminder-${habit.id}`,
  data: {
    habit_id: habit.id,
    habit_name: habit.name
  }
}
```

### AI Prompt Structure

```
Generate a short, motivating notification message for a habit reminder.

Habit: Exercise
Current streak: 12 days
Time until habit: 15 minutes
Phase: early_reminder

Guidelines:
- For early_reminder (15 min): Gentle heads-up, preparation suggestion
- For prep_reminder (10 min): Get ready, gather items needed
- For final_reminder (5 min): Final push, almost time
- For time_now (0 min): It's time, let's go, action-oriented
- Keep it under 60 characters
- Be casual, friendly, motivating
- Don't use "Mirror" in the message
- If streak > 7, mention protecting the streak
- Make it specific to the habit type

Return ONLY the message text, nothing else.
```

### Fallback Behavior

If AI generation fails (network issue, API error):
```javascript
// Falls back to simple message
body: `${habit.name} in ${Math.abs(minutesUntil)} minutes`
```
